## 2024-05-16 - Sample Widget Accessibility & Semantic HTML
**Learning:** React elements without semantic meaning (`<div>`) can cause accessibility issues for screen readers. Uninitialized variables can lead to poor UI experience.
**Action:** Always prefer semantic HTML tags like `<section>`, `<article>`, or `<p>` where applicable. Ensure ARIA labels are added to provide context, and handle missing states with sensible fallbacks.
